# multiroblox-sdk
--

The MultiRoblox SDK is the Source Development Kit, that helps you create your own launcher for Roblox!
*What does it come with?*

--
## It comes with:
- The Cloning System (built on Go)
- Sample UI (Can be edited)
- Other cool stuff you can use to get started on your first Roblox Bootstrapper/Launcher!
